
Motor:
Bafang model todo
8 pole pairs 


Wiring:

- 3 x Digital Input for RPM from  hall sensors
		PG7	D4		GPIO	Input		Phase A
		PI0	D5		GPIO	Input		Phase B
		PH6	D6		GPIO	Input		Phase C

- 1 x PWM  for load transistor control
		PA15	D9		TIM2	CH1


- 4 x Analog inputs for volts, amps, temp, potentiometer  
		
		PA0		A0		ADC3	PIN0		volts
		PF10	A1		ADC3	PIN8		amps
		PF9		A2		ADC3	PIN7		temp
		PF8		A3		ADC3	PIN6		potentiometer
		

- 3 phase rectifier
- Over voltage protection
- Super cap on micro supply line
- 12 v usb car charger
